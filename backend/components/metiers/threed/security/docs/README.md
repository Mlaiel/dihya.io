# docs – Documentation Sécurité (Dihya)

Ce dossier regroupe toute la documentation, les guides, les politiques et les exemples sécurité du projet Dihya.

- `guides/` : Guides techniques, bonnes pratiques, FAQ
- `integration/` : Guides d’intégration, scénarios d’usage
- `policies/` : Politiques de sécurité, modèles, templates
- `samples/` : Exemples de code, helpers, modules sécurité (clé en main)

## Point d’entrée unique

- `index.js` : Importez tous les modules d’un coup :
```js
const docs = require('./index');
docs.guides.<fonction_ou_guide>();
docs.samples.access.generateMfaToken(...);
// ...
```

## Bonnes pratiques
- Utilisez les points d’entrée de chaque module pour vos intégrations
- Inspirez-vous des guides et exemples pour vos propres cas métier
