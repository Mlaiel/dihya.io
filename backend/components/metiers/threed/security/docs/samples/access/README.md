# README – Access Samples

## Structure professionnelle avancée

- `index.js` : Point d’entrée unique pour tous les helpers et samples (clé en main)
- `helpers/` : Helpers ultra avancés pour la gestion des accès (MFA, SSO, provisioning, gestion de session, etc.)
- `samples/` : Exemples d’intégration JS/Python, scénarios concrets, bonnes pratiques
- `__init__.js`, `__init__.py` : Points d’entrée modulaires

## Utilisation
- Importez `index.js` pour accéder à toutes les fonctionnalités d’accès en une seule ligne.
- Importez les helpers dans vos scripts pour accélérer l’intégration sécurité.
- Voir les exemples dans `samples/` pour des scénarios concrets.

## Extension future
